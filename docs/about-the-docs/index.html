How to Use These Docs

This documentation follows the Grand Unified Theory of Documentation, providing four distinct types of content to serve different needs:

Tutorials

Learning-oriented guides for beginners

Step-by-step lessons that guide you through learning Typeflows from the ground up. Perfect for getting started and building foundational knowledge.

Concepts

Understanding-oriented guides for deeper insight

Conceptual discussions that clarify how and why Typeflows works the way it does. Essential for building a deeper understanding of the system.

How-to Guides

Problem-oriented guides for practitioners

Practical recipes that solve specific problems. Ideal when you know what you want to achieve but need guidance on how to do it.

References

Information-oriented technical documentation

Complete technical details, API documentation, and specifications. Your go-to resource when you need precise information whilst working.

The navigation will guide you through learning Typeflows step-by-step, but feel free to jump to any section that matches your current needs.
